# 365-Day Special Calendar

A web application that displays a special day for each day of the year with interactive animations.

## Features

- Interactive calendar with month navigation
- 365 special days with descriptions
- Search functionality
- Today and Random buttons
- Animations for special days
- Recent and upcoming days tracking
- API for data management

## Quick Start

### Using Docker (Recommended)

```bash
# Clone the repository
git clone https://github.com/your-username/365-calendar.git
cd 365-calendar

# Build and run with Docker Compose
docker-compose up -d

# Open in browser
open http://localhost:5000
